# User Feedback Summary

## Source
User testing conducted for Checkpoint 4.

## What Worked
- Clean UI and simple navigation
- Verified student reviews made the platform feel more trustworthy
- Ability to favorite or star apartments was useful

## Pain Points
- Some apartments were missing or had incomplete information
- It was not always clear when login was required versus optional
- The "Write a review" flow was not obvious at first

## Feature Requests
- Add more apartments
- Add filter and search options for price, location, and rating
- Show average ratings at a glance
- Add sorting by most popular or highest rated

## Bugs & Broken Flows
- Occasional login and authentication issues
- Some images did not load correctly
- Review submission did not always save successfully

## Key Quotes
- "This would've saved me so much time before signing my lease"
- "I like that it's only UIUC students, feels more real"
