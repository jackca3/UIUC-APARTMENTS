# Apt.ly - Pre-Launch Workspace

## Project Overview
Apt.ly is a UIUC apartment review platform for students who want honest, student-written information before choosing housing. Verified UIUC students can browse apartments, save favorites, and leave reviews about things like noise, management, and apartment quality. The product is currently live at `https://uiuc-apartments-two.vercel.app/`.

## Team
- Ania

## Tech Stack
- Antigravity for the React-based frontend
- Supabase PostgreSQL database
- Supabase Auth restricted to `@illinois.edu` emails
- Vercel for hosting
- Google Maps / Places API for location details and images
- GitHub for version control

## Live URL
https://uiuc-apartments-two.vercel.app/

## Key User Feedback Themes
- Students trust the verified-review model
- Users want more apartment coverage and better filtering
- Login expectations and review-writing flow need to be clearer
- Reliability issues still exist around auth, image loading, and review submission

## Launch Goal
Soft launch within 1 to 2 weeks, followed by a broader campus push before peak leasing season. Primary success metric: first 50 verified reviews submitted.

## Current Phase
Pre-launch toolkit - building marketing and operations automations for launch readiness.
